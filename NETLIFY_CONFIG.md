## Commandes Utiles

### Lister les variables
```bash
cd frontend
netlify env:list
```

### Ajouter une variable
```bash
cd frontend
netlify env:set VARIABLE_NAME "value"
```

### Vérifier le statut
```bash
cd frontend
netlify status
```

### Lier le projet
```bash
cd frontend
netlify link --id 2d6f74c0-6884-479f-9d56-19b6003a9b08
```
